# 📈 Experimental Results

This directory centralizes the outputs, metrics, and visualizations generated by the Thanos pipeline. It provides a comparative analysis of model performance across different transfer learning scenarios.

## 📁 Output Structure

The results are organized by model type and experiment configuration:

### `NN_TransferLearning/`
- **`Errors/`**: Contains CSV files with performance benchmarks.
  - `NRMSE` and `RMSE` metrics for **Hot Start** (Transfer Learning) and **Cold Start** (Baseline) models.
  - Statistical significance tests (p-values) comparing the two approaches.
- **`Graphs/`**: Visual evidence of model performance.
  - **Density Plots**: Distribution of prediction errors.
  - **Boxplots**: Comparative analysis of variance and outliers across crops.
- **`Models/`**: Persistence layer for the trained models.
  - Best-performing model weights (`.h5` or `.json` formats).
  - Optuna study logs containing the optimal hyperparameters found.
- **`Cleaned_Data/`**: (Optional) Reference snapshots of the feature-engineered data used for the specific run.

---

## 🔍 Key Metrics Tracked

| Metric | Full Name | Significance |
| :--- | :--- | :--- |
| **RMSE** | Root Mean Square Error | Direct magnitude of prediction error in yield units. |
| **NRMSE** | Normalized RMSE | Normalized error allowing for cross-crop comparison regardless of yield scale. |
| **P-Value** | Probability Value | Statistical confirmation that transfer learning improvements are significant. |
